## Atualizando FreeBSD

- portsnap fetch
- portsnap extract
- pkg version -v
- pkg install portupgrade
- portupgrade -arRc #atualiza o sistema todo
_fonte: [portal dos nerds](http://www.portaldosnerds.com.br/?p=1482#comment-166103)_
